# 采用 Codex 式手动功能侧栏布局

阅读器采用类似 Codex 的 macOS 窗口布局：左侧是用户手动展开或收起的功能侧栏，右侧是阅读区，侧栏开关放在红绿灯窗口按钮旁边。这个选择放弃悬停自动显隐和目录浮层，改为让用户明确控制界面暴露程度，更适合摸鱼小窗场景，也更符合 macOS 桌面应用的稳定心智。

**Considered Options**

- 目录浮层按需展开
- 工具入口通过鼠标悬停自动显现
- Codex 式左侧功能侧栏，由红绿灯旁边的侧栏开关手动展开或收起

**Consequences**

- 阅读区始终位于右侧，是主要内容区域。
- 功能侧栏可以承载章节目录、打开文件、阅读设置和老板键设置等入口，但需要保持克制。
- 小窗阅读时，用户可以主动收起功能侧栏，把空间让给阅读区。
- 不做悬停自动展开，避免误触和不稳定的显隐行为。
- 页面原型需要验证侧栏展开态、收起态和小窗态。
