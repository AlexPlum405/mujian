# 章节目录使用浮层侧边栏

此决策已被 `docs/adr/0004-codex-like-manual-sidebar-layout.md` 取代。新的方向采用 Codex 式手动功能侧栏，而不是目录浮层。

章节目录将设计为可展开的轻量浮层侧边栏，而不是大窗口常驻、小窗口收起的响应式侧栏。这个选择优先保护阅读区空间，让目录成为切换章节时短暂出现的工具；代价是用户不能持续看到完整目录，但当前章节状态仍应在主界面中以轻量方式呈现。

**Considered Options**

- 大窗口常驻目录，小窗口收起目录
- 大小窗口都使用可展开目录浮层

**Consequences**

- 阅读区在所有窗口尺寸下都能保持主要空间。
- 主界面需要有清晰但克制的目录入口。
- 当前章节需要在不展开目录时仍可被用户感知。
- 目录浮层的打开、关闭和章节选择反馈需要足够顺手。
