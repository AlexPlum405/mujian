# Triage 标签

工程技能内部会使用 5 个标准 triage 角色。本文件把这些标准角色映射到本仓库实际使用的中文状态。

| 技能内部角色 | 本仓库状态 | 含义 |
| --- | --- | --- |
| `needs-triage` | `待评估` | 需要维护者判断是否要做、怎么做 |
| `needs-info` | `等待补充信息` | 需要提需求的人补充信息后才能继续 |
| `ready-for-agent` | `可交给Agent` | 需求已经清楚，可以交给 agent 独立处理 |
| `ready-for-human` | `需人工处理` | 需要人类亲自实现或决策 |
| `wontfix` | `不处理` | 明确不会处理 |

当技能提到某个标准角色时，使用本表右侧对应的中文状态。
